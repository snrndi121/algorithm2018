# Problem
- There are N people on circlce table. And we will get rid of M_th person. Then this game is starting from (M + 1)th.
- If ther is no person on table, it will be over.
# Condition
- 1 <= M <= N <= 5,000

# Input_1
7 3
# Ouput_1
<3, 6, 2, 7, 5, 1, 4>

# Input_2
5 2
# Ouput_2
<2, 4, 1, 5, 3>
